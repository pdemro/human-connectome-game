# human-connectome

This project, "human-connectome", is a neural connectivity game (001-neural-connectivity-game).

## Technologies Used

*   TypeScript 5.x
*   Node.js 20.x
*   esbuild
*   Vitest

## Project Structure

```
backend/
frontend/
tests/
src/
```

## Getting Started

### Installation

To set up the project, clone the repository and install the dependencies:

```bash
git clone https://github.com/your-username/human-connectome.git
cd human-connectome
npm install
```

### Building the Project

To build the project for production:

```bash
npm run build
```

This will compile the TypeScript code and place the output in the `dist/` directory.

### Development Server

To run the project in development mode with live reloading:

```bash
npm run dev
```

This will start a development server, typically accessible at `http://localhost:8000`.

### Running Tests

To execute the test suite:

```bash
npm test
```

### Linting

To run the linter and check for code style issues:

```bash
npm run lint
```
