# Quickstart: Social Interpreter Game Mode

## Running the Game

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Start the Development Server**:
    ```bash
    npm start
    ```
3.  **Open in Browser**:
    Open your web browser and navigate to `http://localhost:8080`.

## Playing the "Social Interpreter" Mode

1.  From the main welcome screen, click the "Social Interpreter" button to start the game.
2.  Watch the animation of the shapes on the screen.
3.  After the animation finishes, click one of the three buttons at the bottom of the screen to choose the interaction type you believe was shown: "Goal-directed," "Interacting," or "Random."
4.  The game will provide feedback on your choice, and the next round will begin automatically.
